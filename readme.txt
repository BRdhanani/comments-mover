Using comments mover plugin you can move comments between posts and pages in a simple and easy way.

== Description ==
1. Using Comments Mover WordPress Plugin the admin can move any comments from several types of pages pr posts to any other page or post.
2. Admin can have select option for Comments Move in Move Comments menu inside Comments TAB.
3. In TAB there is two drop down selection for source and destination.
4. In source dropdown you can select post or page from which you wants to move comments.
5. In destination dropdown you can select post or page to which you wants to move comments.
6. After selecting both the option click on move comments button and your comments wil be moved.

== Installation ==
Simply select the plugin from the WordPress repository and click the Install button.
